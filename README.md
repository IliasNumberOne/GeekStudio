# 🚀 GeekStudio Company Landing Page

A modern, fully responsive, and interactive landing page built for GeekStudio. Designed with a focus on smooth user experience, dynamic localization, and high performance across all devices.

## ✨ Features

- **🌍 Multilingual Support:** Dynamic language switching between English, Russian, and Kyrgyz using `i18next`.
- **📱 Fully Responsive:** Adaptive layout that works flawlessly on desktops, tablets, and mobile devices.
- **🎨 Smooth Animations:** Engaging UI elements and transitions powered by Framer Motion and Lottie.
- **🎠 Interactive Components:** Responsive carousels and sliders implemented with Swiper.
- **⚡ Optimized Performance:** Modular styling and lightweight structure, ensuring fast load times.

## 🛠 Tech Stack

- **Core:** React, JavaScript, HTML5, CSS3
- **Styling:** Sass (Modular CSS)
- **Localization:** i18next
- **Animations:** Framer Motion, Lottie React
- **UI Components:** Swiper (Carousels)

## 🚀 Getting Started

1. Clone the repository (git clone)
2. Install dependencies (npm install)
3. start the development server (npm start)
